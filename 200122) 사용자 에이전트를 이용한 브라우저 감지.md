# 사용자 에이전트를 이용한 브라우저 감지

window 객체의 navigator 프로퍼티를 사용해 Navigator 오브젝트를 가져올 수 있다.

navigator 프로퍼티의 userAgent 정보를 토대로 현재 브라우저 정보를 알 수 있다.
